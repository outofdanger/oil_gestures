1) VICTORY; включает/выключает курсорный режим. В курсорном режиме сценные жесты (SWIPE/THUMB_UP/…) не действуют — рука управляет мышью (взаимоисключение режимов).
2) FIST; аварийная остановка течения (гасит контроллер -> все струи стоп).
3) POINTING_INDEX; открыть меню-превью выбранного объекта (визуальное, необязательный шаг перед THUMB_UP).
4) SWIPE_LEFT; прокрутка объектов в одну сторону с зацикливанием (по умолчанию выделен первый элемент; каждый свайп переключает на следующий, с последнего — на первый). В зуме контроллера свайп перебирает его кнопки (контекстный набор выбора). SWIPE_RIGHT намеренно отключён: возвратный ход руки распознаётся как противоположный свайп и вызывал скачки — см. gestures/dynamic/model_loader._DISABLED_LABELS.
5) ROTATE_CLOCKWISE / ROTATE_COUNTERCLOCKWISE; регулировка давления контроллера +/- (increase_flow/decrease_flow). Континуальный жест: удержание плавно крутит давление с троттлингом ~0.25с; работает при запитанном контроллере.
6) THUMB_UP (решили вместо ok_sign); активировать выбранную деталь — действие зависит от типа: вентиль -> открыть/закрыть, манометр/заглушка -> снять/установить, рычаг/дверца/крышка/кнопки контроллера -> их клик-логика (_dispatch_click).
7) SQUEEZE; приблизиться (focus) к выбранному узлу — контроллер / уровнемер / манометр.
8) RELEASE; вернуться в общий вид (unfocus).
