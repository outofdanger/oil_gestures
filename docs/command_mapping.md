1) VICTORY; включает/выключает курсорный режим. В курсорном режиме сценные жесты (SWIPE/THUMB_UP/…) не действуют — рука управляет мышью (взаимоисключение режимов).
2) FIST; НЕ жест (аварийку жестами убрали — пересекается со SQUEEZE-зумом). Аварийный стоп: панель «АВАРИЙНЫЙ СТОП» + кнопка контроллера (controller_big_button) + меню.
3) POINTING_INDEX; открыть меню-превью выбранного объекта (визуальное, необязательный шаг перед THUMB_UP).
4) SWIPE_LEFT / SWIPE_RIGHT; прокрутка объектов вперёд/назад с зацикливанием (по умолчанию выделен первый элемент). В зуме контроллера свайп перебирает его кнопки (контекстный набор выбора). Оба направления снова включены: возвратный ход руки (ложный противоположный свайп) гасится направленным лок-аутом — см. gestures/dynamic/model_loader._OPPOSITE_LABELS (directional_lockout_seconds).
5) ROTATE_CLOCKWISE / ROTATE_COUNTERCLOCKWISE; регулировка % открытия ВЫБРАННОГО вентиля (set_position, шаг 10%). Через цепочку это регулирует давление/поток (actual_pressure = target * min_percent/100). Континуальный жест: удержание плавно крутит вентиль с троттлингом ~0.25с. Если выбран не вентиль — подсказка. (Глобальное давление контроллера регулируется отдельно — его кнопками +/-.)
6) THUMB_UP (решили вместо ok_sign); активировать выбранную деталь — действие зависит от типа: вентиль -> открыть/закрыть, манометр/заглушка -> снять/установить, рычаг/дверца/крышка/кнопки контроллера -> их клик-логика (_dispatch_click).
7) SQUEEZE; приблизиться (focus) к выбранному узлу — контроллер / уровнемер / манометр.
8) RELEASE; вернуться в общий вид (unfocus).
